What is JavaScript?

JavaScript is a programming language that is used to make web pages interactive. It is a client-side scripting language that is used to create dynamic web pages. It is a lightweight, interpreted programming language with first-class functions, most known as the scripting language for Web pages, but it's used in many non-browser environments as well.
